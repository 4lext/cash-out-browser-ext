Troubleshooting

Extension Not Working?

  • Check Console: Open DevTools (F12) and check for errors
  • Reload Extension: Go to chrome://extensions and click the reload icon
  • Check Permissions: Make sure the extension has access to cdn.jsdelivr.net
  • Network Issues: Check if you can access the CDN directly
